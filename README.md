# VK Access Token — Kate Mobile

Простая статическая страница для получения пользовательского access token ВКонтакте через приложение **Kate Mobile** (app_id `2685278`), с полным набором прав (`scope=1073737727`).

## Как работает
- Кнопка открывает официальный `https://oauth.vk.com/authorize` (токен идёт напрямую от VK, страница его не получает).
- После «Разрешить» VK редиректит на `oauth.vk.com/blank.html#access_token=...`.
- Вставляешь этот URL в поле на странице — токен извлекается и копируется.

## ⚠️ Важно
- **Неофициальный способ.** Токен Kate Mobile живёт ~24 часа и **не обновляется** (implicit flow, без refresh_token) — по истечении получай заново.
- Храни токен как пароль.
- Возможен бан аккаунта VK за активность через неофициальный клиент.
